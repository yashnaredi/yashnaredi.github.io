Email – Exercise 2

Inbound Email Flow with Auto-Reply Using Genesys Cloud Default Domain

Objective

Create an inbound email Architect flow that includes an automatic reply to customers while routing emails to a queue.

Steps

  1. Set Up Email Address
    • Use the existing or create a new email address in Contact Center > Email with the @mypurecloud.com domain
    • Configure routing to use an Architect flow instead of direct queue routing
  2. Create Architect Inbound Email Flow
    • Navigate to Architect and create a new inbound email flow
    • Add a Send Email action to send an automatic acknowledgment or auto-reply to the customer
    • Customize the auto-reply message to confirm receipt and expected response time
    • Route the email to the appropriate queue for agent handling
    • Optionally add conditions, such as keywords or time-based routing
    • Save and publish the flow
  3. Assign Flow to Email
    • Return to the email address configuration
    • Set routing to the newly created Architect inbound email flow
    • Save changes
  4. Test the Setup
    • Send test emails to the configured address
    • Verify that an auto-reply email is received
    • Confirm email routes to the assigned queue and agents receive notifications
