How to Configure GPE (Genesys Predictive Engagement)

Prerequisites

  • Ensure that Predictive Engagement is enabled in your Messenger Configuration settings.

Step 1: Create Segments

  • Define key user attributes and behaviors to segment your website visitors.
  • Optionally, assign colors to segments to easily differentiate them.

Step 2: Create Outcomes (Optional)

  • Outcomes help track goal completions and are useful for reporting and analytics.
  • They can be tied to user actions such as form submissions, page visits, or engagement milestones.

Step 3: Create an Action Map

  • You can configure:
    • Content Offers (like coupons, forms, etc.)
    • Messenger Popups to proactively engage users

Step 4: Set Triggers

  • Use previously created segments to activate the action map
  • Alternatively, use user activity such as page scrolls or visit duration as triggers

Step 5: Optional - Add Constraints

  • Configure outcome triggers and offer constraints to control how and when an action appears

Step 6: Build Your Content Offer

  • Navigate to the Action Library
  • Create a content offer (image, text, button, link, etc.) with targeting and behavior settings

Step 7: Whitelist Your Domains

  • Make sure all domains where GPE should operate are added to your whitelist

Step 8: Monitor Activity

  • Use the Live Now page to monitor visitor behavior and action map triggers in real time
  • Evaluate the effectiveness of segments, outcomes, and offers continuously
