How to Set Up Copilot

Prerequisites

  • Ensure correct org permissions are in place – Permissions Guide
  • Enable AI Tokens – AI Tokens Info

Setup Instructions

Navigate to Contact Center > Agent Copilot and follow these steps:

  1. Set up Knowledge Base
    • Add relevant articles
    • Save your knowledge base
  2. Set up Copilot
    • Use the Knowledge Base from Step 1
    • Toggle Answer Generation and Answer Highlights as needed
    • Configure NLU and knowledge thresholds
  3. Assign Copilot to Queue
    • Use an existing queue or create a new one
    • Update agent assignments for the queue
    • Assign Copilot to yourself and team members

Testing

  • Go on-queue and accept an incoming conversation
  • Use the Lightbulb (solid fill), Picture, or Picture-and-Talk icon to interact
  • Copilot will provide suggested answers based on your configuration
