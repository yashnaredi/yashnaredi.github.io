Messenger Deployment & Widget Configuration

Messenger Configuration

Navigate to Message > Messenger Configuration and follow these steps:

  • Configure desired Messenger appearance
  • Add logo
  • Configure widget color
  • Select default language
  • Set Messenger alignment
  • Assign an avatar and name for your bot
  • Optionally configure:
    • Co-browse
    • Genesys Predictive Engagement (GPE)
    • Knowledge portal

Messenger Deployment

Navigate to Message > Messenger Deployment and follow these steps:

  • Select the previously configured Messenger setup
  • Whitelist the desired domains
  • Select the Architect flow associated with this deployment
  • Save the deployment snippet
  • Embed the snippet into your website
