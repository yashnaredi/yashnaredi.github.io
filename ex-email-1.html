Email – Exercise 1

Basic Email Setup Using Genesys Cloud Default Domain (mypurecloud)

Objective

Set up a basic email address in Genesys Cloud using the default domain (mypurecloud.com) and route incoming emails directly to a queue.

Steps

  1. Create Email Address
    • Go to Contact Center > Email in Genesys Cloud
    • Click “Add Email Address” and use the default domain (@mypurecloud.com)
    • Fill in the From Name and Reply-To email fields as desired
    • Save the new email address
  2. Configure Routing
    • In the email configuration, select “Always route to this flow/queue” option
    • Choose an existing queue for email handling or create a new one
    • If routing to a queue, optionally configure skills, language, and priority
    • Save the routing settings
  3. Test Basic Routing
    • Send a test email to the configured @mypurecloud.com address
    • Verify the email is routed directly to the assigned queue
    • Check agent availability and email assignment in Genesys Cloud
