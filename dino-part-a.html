Part A: Setup & Configuration

Prerequisite: Set up access to GenPSTest environment for this task.

Objective: Dino Park is a website where users get answers from agents about support, sales, and science. Your task is to configure inbound flows for these 3 lines of business (LOB):

  • Science
  • Sales
  • Support

Communication Channels:

  • Web Messaging
  • Email
  • Chat (Facebook/X)

Trainees are expected to configure queues and schedule groups for each LOB.

Display estimated wait time for chats using the existing Data Action "Get Estimated Wait Time". Show wait time in 5-minute intervals (e.g., "Your estimated wait time is between 5 to 10 minutes"). Convert output from seconds to minutes.

Web Messenger Configuration & Deployment

  • Customize widget colors, logo, and bot persona to match Dino Park branding.
  • Note: Knowledge portal configuration is not required for this exercise.
  • Deploy widget with provided code snippet.
  • Test on Dino Park website using Tamper Monkey (Chrome extension).

Testing Web Messenger Routing

  • Use database.set to pass keys (science, sales, support) for routing.
  • Handle off-hours chats by routing to fallback queue or Support queue.

Email Configuration

  • Configure domains/subdomains for each LOB to receive emails.
  • Set up LOB-specific auto-responses.

Chat (Facebook/X) Setup

  • Configure channel via existing API integration.
  • Connect inbound message flow to handle chats.
